# TwitterBoTTweet
Twitter-Tweet-Bot

# Installation:

Requires Python 3.*<br>
Requires Selenium intalled by ```pip3 install selenium```

# Use:

Run it and a Tweet with the Introduced text will appear in the account specified<br>
Example: https://twitter.com/asdasd91937188
